VIA RÁDIO

INTERNET RURAL

A transmissão de internet via rádio consiste na distribuição de sinal a partir de uma torre que é recebida por antenas instaladas que transmitem o sinal.

 

Este tipo de conexão é desvinculado de linhas telefônicas e pode oferecer altas velocidades quando há uma transmissão de qualidade como é a da GOX.

 

A internet via rádio dispensa a utilização de um modem, sendo necessária apenas a instalação de uma antena.
